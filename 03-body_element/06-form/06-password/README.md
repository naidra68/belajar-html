# Form Password

Penulisan input password sama halnya dengan input type `textbox`. Perbedaan-nya inputan password (sesuai nama) digunakan untuk inputan yang memerlukan kerahasiaan. Dengan menggunakan input password ini, semua yang kita tulis akan disamarkan dengan simbol bulat atau bintang.

berikut contoh penulisan-nya

```html
<p>Password : <input type="password" name="password_user"></p>
```

Attribut `name` disini kita gunakan hanya untuk mendefinisikan bahwa inputan ini password user, fungsi sebenar-nya akan terlihat jika sudah memasuki server.
