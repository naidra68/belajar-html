Istilah-istilah dalam git:

Git Status
Git Status digunakan untuk melihat staging area. Semua file untrack akan tampil di staging area sebelum di tambah dan di commit.
Git Add
Git Add digunakan untuk menambahkan file yang ada di staging area. Ketika git add . diketik maka semua file di staging area akan ditambahkan.
Git Commit
Git Commit digunakan untuk menyimpan file yang ada di staging area. Ketika git commit diketikkan maka semua file akan keluar dari staging area dan akan disimpan.
Git Push
Git Push digunakan untuk mengalihkan file dari local ke remote. Ini cukup rumit dipahami karena kita bekerja tidak hanya pada git namun juga remote git seperti github, gitlabs, dan bitbucket.
