# Block Level Element

Block level element adalah tag HTML yang ditampilkan full selebar halaman. Contoh tag yang bersifat block level element adalah tag `<p>` dan tag `<h1>`. Kedua tag ini akan full ditampilkan selebar halaman dan akan membuat baris baru jika terdapat tag lain dibawahnya. Selain itu, tag sejuta umat yaitu tag `<div>` juga masuk ke dalam block level element.

# Inline Level Element

Inline level element adalah tag HTML yang ditampilkan mengikuti isi konten dari element tersebut dan tidak akan membuat baris baru. Contoh tag yang bersifat inline level element adalah tag `<img>` dan tag `<em>`. Kedua tag ini akan mengikuti konten-nya dan tidak membuat baris baru. Selain itu, tag sejuta umat yaitu tag `<span>` juga masuk ke dalam inline level element.
